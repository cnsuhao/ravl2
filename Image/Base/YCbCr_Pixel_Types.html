In RAVL, YCrCb is used to refer to the colour spaces used in digital video standards. There are currently two main YCbCr colour spaces:

  • Standard definition television, described in ITU Rec 601. In RAVL, these are the "...YCbCrBT601..." pixel types.
  • High definition television, described in SMPTE 274 and EBU - TECH 3299. These are appearing "soon" in RAVL.

The "...YUV..." classes and conversions on this page are incorrectly named YCrCb ones, and should disappear soon...

N.B. The 422 pixel classes contain one Y component, and one Cb|U or Cr|V component, according to the column number in the image.
