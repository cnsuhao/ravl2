There are currently two main YCbCr colour spaces:

  • Standard definition television, described in ITU Rec 601
  • High definition television, described in SMPTE 274 and EBU - TECH 3299

N.B. The 422 pixel classes contain one Y component, and one U or V component, according to the column number in the image.
