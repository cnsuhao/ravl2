There is much confusion about the difference between YUV and YCbCr. Within RAVL, the only notable difference is in the matrix values used to convert between YUV (or YCrCb) and RGB. The term YUV is frequently used when YCrCb is meant. We have introduced separate pixel classes to help people produce the correct colour rendering of their images.

  • The YUV/RGB conversion matrices traditionally used by the image processing community are based on those used in analogue colour television.
  • The YCrCb/RGB conversion matrices are based on those used in digital television sources.
