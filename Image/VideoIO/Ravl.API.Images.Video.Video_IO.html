The following classes implement video file handling built into RAVL. Normally you don't use them directly, but via the Sequence IO mechanism described here.

See the "Subtopics" (if any) for other video formats. These will be mainly from the contrib set; however anything that appears in the Subtopics has been installed on the system already.
