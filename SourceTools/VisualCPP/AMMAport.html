AMMA Visual C++ Port
Centre for Vision, Speech & Signal Processing

This page is concerned with the ongoing AMMA port to VISUAL C++. By following the simple instructions below you can write code in Visual C++ and link to the AMMA libraries. You can also help port the remaining code to Visual C++.

Using AMMA from within Visual C++

All the AMMA header files, libraries and source code are in the directory tree:/vol/vssp/local/VisualCPP. To use AMMA, you first have to make a soft link to this directory in your home directory. This can be done by typing the following command on your unix machine.

ln -s /vol/vssp/local/VisualCPP ~/soft/VisualCPP

Next when you are in Windows you have to map your home directories as drive h:. This can be done easily in Windows explorer through the tools menu. Finally, in Visual C++ open the Tools/Options window. Tab the Directories tag and add the two following standard paths.

h:/soft/VisualCPP/inc ... for include files

h:/soft/VisualCPP/lib/debug ... for debug libraries

You can now include amma header files in the usual way. When you link an executable, the library is: AMMAported.lib I am working on a solution.

Also note that not all libraries have been ported. This file lists all files that have been ported so far. Note, that only the threaded libraries have been built.

Porting AMMA

If you want to attempt to port some libraries which are not yet ported read on. The first step is to make a mirror copy of the AMMA source code.

am mirror MIRROR="/your/mirror/dir"

Next you need to use a program which converts all the defs.mk files to the relevant workspace files. This is doVCPP.

doVCPP -i /your/mirror/dir -o /your/out/dir -dp your/out/dir -d h -ex

In the output directory you will see new directories made. These contain all the AMMA include and src files. Also created are the project files. One for each library and each executable. Choose the library you want to port, load it into the workspace and try to compile it.

Once you have successfully ported it, let me know and I will add to the ported list.
