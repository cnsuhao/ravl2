These functions are designed to provide just enough functionality to ensure that the core library is thread safe. Full threading functionality will be provided elsewhere.

These functions are transparently enabled when the RAVL threads library RavlThreads is linked into a program. Id 0 is a general system lock.
Id 1 is used for locking the random number generator.
Id 2 is used for non-reentrant 'C' library calls.
Id 3 is reserved for use in RavlLogic.
