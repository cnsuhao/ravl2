All objects which derive from DPPortC , inherit an attribute mechanism for data processing objects. The Method 'bool GetAttrList(DListC & nlist) const' will append the list of available attributes to nlist. If the data processing object is in a class hierarchy, the 'GetAttrList' method will return all attributes from parent classes also.

In order to determine the current value of a given attribute, the overloaded methods :-

  • StringC GetAttr(const StringC & attrName)
  • bool GetAttr(const StringC & attrName,StringC & attrValue)
  • bool GetAttr(const StringC & attrName,IntT & attrValue)
  • bool GetAttr(const StringC & attrName,RealT & attrValue)

are provided. 'attrName' is a string containing the name of the attribute that you wish to find the value for. The first form of this method returns a string containing the attribute value, a null string is returned if the attribute name is not found. The other forms of this method return true or false depending if the attribute was found. The value is passed by reference into attrValue.

Attributes may be set with the following methods :-

  • bool SetAttr(const StringC & attrName,const StringC & attrValue)
  • bool SetAttr(const StringC & attrName,const IntT & attrValue)
  • bool SetAttr(const StringC & attrName,const RealT & attrValue)

these methods return true if the 'attrName' was valid for objects of this type.
