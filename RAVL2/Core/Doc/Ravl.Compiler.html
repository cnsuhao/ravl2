You may need to be logged in as root for this. These instructions worked some time ago. After downloading from the Gnu site, the installation went like this: 

bunzip2 gcc-4.0.2.tar.bz2
tar -xvf gcc-4.0.2.tar
mv gcc-4.0.2 gcc-4.0.2-src
cd gcc-4.0.2-src/
./configure --prefix=/usr/bin/gcc-4.0.2 --enable-languages=c,c++
make bootstrap
setenv LD_LIBRARY_PATH /vol/gcc/lib:/vol/ai/lib:${LD_LIBRARY_PATH}
make install
cd ..
