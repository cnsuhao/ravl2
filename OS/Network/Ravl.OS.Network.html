There are currently 3 levels of networking support in RAVL.

Low level

SocketC - This class provides low level socket operations associated TCP/IP and DNS networking. It hides the platform specific implementation details.

Medium Level

NetIStreamC & NetOStreamC - These allow a network stream to treated much like a file. You can use these to send data across a connection as text with the standard C++ '>>' and '<<' stream operators or wit BinIStreamC and BinOStreamC as machine independant binary. NetEndPointC - Provides a mechanism for calling methods of a class across the network.

High level

NetPort This class allows streams of data to exported across the network. e.g. This can be used for transporting the results of image processing, or the raw images from a grabber card.
