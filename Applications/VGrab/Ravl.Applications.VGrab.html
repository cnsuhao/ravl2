! author="Lee Gregory" ! docentry="Ravl.Applications.VGrab" : Video Frame Grabber Video Frame Grabber

This program captures video from grabber cards. The grabber device can be specified using RAVL virtual files and defaults to the ClipStationPro driver.

Usage: 

 
   -g (false) [false] Get current timecode
   -v (false) [false] Be verbose ?
   -start (now) [now] The starting timecode [HH:MM:SS:FF , now]
   -end () [] The ending timecode [HH:MM:SS:FF]
   -out (tmp1.4cif) [tmp1.4cif] The output filename for capture device 1 (.cif, .4cif,  .ppm, .avi, .jpg )
   -dev (@CSP:PCI,card:0) [@CSP:PCI,card:0] The capture device (See RAVL virtual files for more details )
   -n (25) [25] Duration of capture (number of frames to be grabbed), NB Cannot be used with real timecodes
   -step (1) [1] Capture every nth frame
   -help (false) [false] Print usage information.
 
Dependancies:
   -end and -n are mutually exclusive
